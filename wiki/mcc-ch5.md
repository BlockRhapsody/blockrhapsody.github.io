# 第五章 目标选择器

> ⚠️ 本章内容的学习必须配合游戏实操！


## 5.1 目标选择器的作用

目标选择器让命令能够动态地选中一个或多个玩家或实体作为操作对象。

例如：`/give @a diamond_sword` — 给予所有玩家一把钻石剑。
`/kill @e[type=zombie]` — 杀死所有僵尸。


## 5.2 基础目标选择器

| 选择器 | 含义 |
| :--- | :--- |
| `@p` | 最近玩家 |
| `@r` | 随机玩家（可配合`type`选择随机实体） |
| `@a` | 所有玩家 |
| `@e` | 所有实体 |
| `@s` | 命令的执行者 |
| `@n` | 最近实体（基岩版独有） |
| `@initiator` | 与NPC交互的玩家（基岩版独有） |

> ⚠️ 中国版没有`@initiator`选择器，因为中国版没有NPC交互功能。


## 5.3 目标选择参数

### 坐标与距离参数

| 参数 | 说明 |
| :--- | :--- |
| `x`、`y`、`z` | 指定坐标原点 |
| `r` | 最大半径 |
| `rm` | 最小半径 |
| `dx`、`dy`、`dz` | 长方体区域尺寸 |

### 计分板与标签参数

| 参数 | 说明 |
| :--- | :--- |
| `scores` | 筛选具有指定计分板分数的实体 |
| `tag` | 筛选具有（或不具有）指定标签的实体 |

### 实体类型参数

| 参数 | 说明 |
| :--- | :--- |
| `name` | 筛选具有指定名称的实体 |
| `type` | 筛选指定类型的实体 |
| `family` | 筛选指定族的实体（基岩版独有） |

### 数量参数

| 参数 | 说明 |
| :--- | :--- |
| `c` | 控制选择的目标数量（正数取前N个，负数取后N个） |

### 常用组合示例

选择半径10格内的所有玩家
@a[r=10]

选择半径5格到10格之间的所有僵尸
@e[type=zombie,rm=5,r=10]

选择背包内有钻石剑的玩家
@a[hasitem={item=diamond_sword}]

选择计分板"等级"分数大于10的玩家
@a[scores={等级=10..}]

选择距离最近的生存模式玩家
@p[m=survival]

## 5.4 性能考量

不当使用的目标选择器可能造成明显的性能开销：

### 优化建议

1. **优先使用更精确的选择器**：能用`@a`就不用`@e`
2. **控制搜索范围**：尽量使用`r`或`dx/dy/dz`缩小范围
3. **避免高频执行大范围`@e`**
4. **使用`c`参数限制数量**
5. **用`tag`预筛选**：预先添加标签，避免每次都靠`type`匹配
